## 三级 · 操作手册 (OPERATIONS)

### 3.1 每日检查项

```
⏰ 00:00 CST → 费率结算 + 日报
  □ 检查PnL
  □ 检查清算距离
  □ 检查挂单状态
  □ 是否需要调仓

⏰ 08:00 CST → 费率结算 + 15min巡检
  □ 网格是否正常运行
  □ 有无异常触发

⏰ 16:00 CST → 费率结算 + 午检
  □ 止盈有没有成交
  □ 是否需要补充网格层

异常触发:
  □ 清算<8% → 自动减仓
  □ 网格全成交 → 自动重新部署
  □ 费率反转 → 调整多空比
```

### 3.2 指令集

```
DS-0 接收以下指令:
  "状态"     → 输出完整持仓/挂单/风险报告
  "日报"     → 输出今日PnL/费率/操作统计
  "调仓"     → 重新评估并调整所有参数
  "紧急"     → 输出风险警告 + 建议操作
  "复盘"     → 输出今日完整复盘
```
