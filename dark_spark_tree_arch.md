## 数据流循环 (30分钟闭环)

```
[DS-0 Root] 每30分钟:
  1. 读 bus/income_30d.json (最新损益)
  2. 读 bus/trade_logs/ (笔数·胜率·夏普)
  3. 读 bus/market.json (当前行情·费率·多空比)
  ↓
  4. 判断: 需要深度分析吗?
     ├─ 是 → 唤醒GPT [K线+策略审计]
     └─ 否 → 自判断: 微调/持有/清仓
  ↓
  5. 写 bus/commands.json (执行指令)
  ↓
[执行层] 读命令 → 下单/调参
  ↓
[执行层] 写回结果到 bus/exec_results.json
  ↓
[进化引擎] (每天04:00)
  读一天数据 → 淘汰亏损策略 →变异新参数 → 推优实盘
  ↓
[回到1]
```

## 30天损益 → 直接决定下一阶段策略
