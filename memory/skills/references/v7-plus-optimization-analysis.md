# V7+ 7天优化分析 (2026-05-09)

## 数据基础
- 10个币种 (GWEI/SPORTFUN/DEEP/ALL/SIREN/FIGHT/COLLECT/BULLA/SPY/ORCA)
- 7天15分钟K线 (672根/币种)
- 50条真实资金费率记录/币种
- 140种参数组合全面扫描

## 关键发现

### 1. 止损=关闭最优
- 2xATR产生1134笔交易/7天，大部分是止损→重开循环
- ATR_STOP_MULT=0.0时胜率54.5%，远高于2.0时的36.8%
- 关闭止损后每笔平均持有时长从~3分钟提升到~20小时

### 2. EMA5/20交叉止盈=纯噪声
- 小币种1分钟K线的EMA交叉完全跟随随机波动
- 去掉EMA止盈后胜率从32%→54%
- 改用24h固定出场效果最佳

### 3. 24h=最优持仓
- 测试区间: 2/4/6/8/12/16/20/24h
- 前6小时胜率仅40.9%，均亏-1.66%
- 8h: 54.5%胜率, +0.24%
- 12h: 68.2%胜率, +2.84% (最优单日)
- 24h: 54.5%胜率, +2.45% (7天最优综合)

### 4. 费率符号方向 ≈ 固定做多 (偏牛市)
- 但费率符号方向符合套利逻辑，保留
- 固定做多此时表现更好但逻辑不成立
