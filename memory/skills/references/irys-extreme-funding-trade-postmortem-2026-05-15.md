## 教训

### 1. 极端费率 ≠ 入场信号

-1.2%费率只说明空头在死扛，不说明价格会继续涨。**如果价格已经涨了+27%，再好的费率也是追高。**

### 2. 入场时机三要素

```
✅ 好费率 (-0.5%+)
✅ 价格在启动初期 (<+10%)
✅ 量在放大但不是顶峰

❌ IRYS案例: 好费率 ✅ + 高位追多 (-27%) = 亏损
```

### 3. 高位极端费率的陷阱

极端费率（绝对值>1%）通常出现在两种场景：
- **场景A**: 价格刚从底部启动，空头大量做空 → ✅ 逼空好机会
- **场景B**: 价格已经拉了很多，空头不信邪继续加空 → ⚠️ 可能双爆（空头爆+多头回撤）

IRYS是场景B。我们误判成场景A。

### 4. 收缩策略有效

用户说"收缩"（减半仓）在实际操作中是正确的：
- 卖一半 → 亏损从$36减到~$18
- 留一半 → 还有上涨暴露，继续吃费率
- 释放$25保证金备用

### 5. 止损设置原则

| 条件 | 止损距离 |
|:----|:--------:|
| 正常建仓 (趋势初期) | 3-5% |
| 高位追多 (+15%+) | 2-3%,或直接不进 |
| IRYS实际止损 | -6.5% ($0.066) — 太大,应该在进场后立即收窄到3% |

## 改进后的入场检查清单

```python
def check_extreme_funding_entry(symbol, current_chg_24h, fr_pct):
    \"\"\"极端费率入场前检查\"\"\"
    warnings = []
    
    # 1. 价格位置
    if current_chg_24h > 15:
        warnings.append(f\"价格已涨{current_chg_24h:.0f}%,追高风险大\")
    elif current_chg_24h < -15:
        warnings.append(f\"价格已跌{current_chg_24h:.0f}%,可考虑抄底\")
    
    # 2. 费率性质
    if fr_pct < -0.5:
        warnings.append(f\"极端负费率{fr_pct:.1f}%: 场景A(底部)还是场景B(高位)?\")
    elif fr_pct > 0.5:
        warnings.append(f\"极端正费率{fr_pct:.1f}%: 多头过热信号\")
    
    # 3. 涨跌前价格 (看1h前价格,不是24h前)
    # 如果从底部$0.05涨到$0.07(+40%),即使只是+20%也危险
    
    return warnings
```

## 费率和方向的权重

**本交易给铁律: 方向权重 > 费率权重。**

```
方向风险 × 仓位大小 × 杠杆 = 潜在亏损
费率收入 × 持有时间 = 潜在收益

当 方向风险 > 费率收入 × 3天时 → 不开仓
```

本交易中:
```
方向风险: $0.0706 → $0.065 (止损) = -$65
费率收入: $25/天 × 3天 = $75

看起来平衡,但方向亏损1天就到了,费率要3天才cover
```
