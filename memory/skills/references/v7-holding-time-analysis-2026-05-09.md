# V7持仓时间优化分析 (2026-05-09)

## 数据基础
- 日期: 2026-05-09
- 分析脚本: `autonomous/analyze_holding_time.py`
- 结果文件: `analysis/holding_time_analysis.json`
- 数据源: V7日志 + 币安真实K线（22笔开仓, 4批次, 10个币种）
- 分析方法: 用今天实际开仓时间和价格, 用K线模拟不同持仓时间的收益

## 核心结论

### 🏆 最优: 固定持仓12小时

| 持仓时间 | 胜率 | 均收益 | 盈亏比 |
|---------|:---:|:------:|:-----:|
| 0.5~6h | 40.9% | -1.66% | 0.41 ❌ |
| 8h | 54.5% | +0.24% | 1.60 |
| **12h ★** | **68.2%** | **+2.84%** | **3.67** |
| 16~24h | 68.2% | +2.84% | 3.67 |

### 现有ATR+EMA策略对比
- 胜率: 9.1% ❌
- 均收益: -1.74%
- 盈亏比: 0.02
- 几乎全部触发止损, 形同虚设

### 建议
- 将V7的止盈逻辑从EMA5/20交叉改为: 固定持仓12小时全平
- 保留2xATR止损作为极端兜底（价格反向超2xATR立即止损）
- 12h后无论盈亏, 强制平仓

## 分析脚本
```bash
python3 autonomous/analyze_holding_time.py
# 输出: analysis/holding_time_analysis.json
```

## 分析脚本核心逻辑
1. 从日志解析每笔开仓(时间, 币种, 方向, 价格)
2. 从币安拉取该币种K线数据
3. 对每个持仓模拟N小时后的价格
4. 比较不同N的胜率/收益/盈亏比
5. 同时模拟现有ATR+EMA止损止盈规则的结果
