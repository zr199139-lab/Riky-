## Key lesson captured

The rapid-iteration cycle (parameter sweep → review → discard) leaves more intermediate files than production code. A 3-tier classification (safe-delete / archive / keep) prevents accidental deletion of running Bot source files while keeping the directory navigable.

The `rm` command was blocked by user (`BLOCKED: Command denied by user`), confirming that user wants **review-before-delete** workflow, not automated cleanup.
