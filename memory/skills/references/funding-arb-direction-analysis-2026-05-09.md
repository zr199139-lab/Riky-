# 费率套利方向判断分析 (2026-05-09)

## 背景
V3之前的费率套利Bot用费率符号决定方向（费率为负做空，费率为正做多）。
这个假设等价于「费率为谁付费谁的方向就错」——实证数据显示这是错误的。

## 数据实证

### 费率符号方向正确率：50%（随机水平）
15笔开仓，10笔有1h后K线数据：
- 09:01 ACE 做空 -6.61% 对
- 09:01 ALICE 做空 -1.28% 对
- 09:01 DEEP 做空 -3.80% 对
- 09:01 SIREN 做空 +1.33% 错
- 09:01 QQQ 做多 -0.06% 错
- 10:03 ACE 做空 -1.72% 对
- 10:03 ALICE 做空 -0.71% 对
- 10:03 DEEP 做空 +0.44% 错
- 10:03 SIREN 做空 +0.70% 错
- 10:03 QQQ 做多 +0.00% 错

结论: 5/10 = 50%。跟抛硬币没区别。

### EMA9方向判断更差：20%（2/10）
ACE 09:01 EMA向上做多 vs 实际跌6.61%（错）
根因：1h级别涨跌接近随机，任何技术指标都无预测能力。

### 4h趋势方向一致性 = 60%（3/5）
- ACE: 费符空 vs 4h趋势空 一致
- SIREN: 费符空 vs 4h趋势多 相反
- ALICE: 费符空 vs 4h趋势空 一致
- DEEP: 费符空 vs 4h趋势空 一致
- QQQ: 费符多 vs 4h趋势空 相反

一致率3/5=60%。费率方向与4h趋势方向一致时准确率更高。

## V4修改方案
1. 方向判断：4h/24h趋势取代费率符号
2. 仓位：ATR决定（不跳过，不过滤）
3. 出场：去固定止盈，加趋势反转退出
4. 止损：保留绝对价格-8%
