## 三、学习路线图

### Phase 1: 数据积累（第1-4周）
- [ ] 每次截图收入档案
- [ ] 追踪其持仓变化（何时平仓、何时加仓）
- [ ] 记录胜率/ROI变化趋势

### Phase 2: 模式识别（第2-8周）
- [ ] 识别其入场信号（突破追？回调接？）
- [ ] 识别其出场规则（止盈点？止损容忍度？）
- [ ] 识别其选币逻辑（热点？基本面？技术面？）

### Phase 3: 小资金模拟（第4-12周）
- [ ] 用类似手法做$50-100模拟仓
- [ ] 对照真实交易员的操作做比对
- [ ] 逐步形成自己的风格

### Phase 4: 实盘适配（第8周后）
- [ ] 将学到的模式适配到小资金实盘
- [ ] 小资金需要更紧止损（不能学-29%硬扛）
- [ ] 形成自己的"聪明钱融合策略"
