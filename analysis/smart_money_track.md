## 四、核心可学技术

### 从Trader H学：趋势持有
```
他SOL从低位入场到+147%不卖
ETH从低到+123%不减仓
今天才开始减TON和ETH锁利
→ 小资金可以学：看对方向就让趋势跑，别赚3%就跑
```

### 从Trader E学：极致盈亏比
```
胜率29.81%，但赚$1M
单笔BTC赚+923%
→ 10笔交易亏7笔，3笔赚的全是大肉
→ 小资金打法：小止损($5)，让盈利跑到$30+
```

### 从Trader G学：多空对冲
```
唯一做ETH空单的交易员
对冲了AIA的多单亏损
→ 当大盘不确定时，多空都开，吃方向的那一边
```
