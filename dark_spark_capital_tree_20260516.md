## 二级 · 专业级风控体系 (PROP FIRM STANDARDS)

### 2.1 单笔风控 (Per-Trade Risk)

| 规则 | 标准 | 来源 |
|:----|:----|:----|
| 单笔风险 | ≤账户总额的2% | Prop Firm铁律 |
| 止损距离 | 1.5 × ATR(14) 自适应 | 机构标准 |
| 杠杆上限 | 10x (硬编码) | 量化机构 |
| 保证金 | $5-30/笔 | 小资金保护 |
| 止损执行 | 不设止损不开单 | 铁律 |

### 2.2 日/周风控 (Daily/Weekly Risk)

| 规则 | 阈值 | 动作 |
|:----|:----|:------|
| 日亏上限 | -$30 | 停开新仓24h |
| 回撤>10%追踪 | 从账户峰值算 | 警告+减仓建议 |
| 回撤>20%追踪 | 从账户峰值算 | **停所有交易** |
| 总敞口 | ≤50%资金 | _validate_trade闸8拦截 |

### 2.3 仓位计算 (ATR Position Sizing)

```
仓位 = 风险预算 ÷ (ATR × 1.5)

示例 ($690账户):
  风险预算 = $690 × 2% = $13.80 (单笔最大亏损)
  ETH ATR(1h) = $33 → ATR% = 1.5%
  止损距 = 1.5 × $33 = $49.5 (2.2%)
  建议保证金 = min($30, max($5, 风险预算/止损距×杠杆))
```
